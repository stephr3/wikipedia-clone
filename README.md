# Wikipedia Homepage Clone
## *By Stephanie Gurung*

A clone of wikipedia's home page.

## Technologies Used

* HTML<br>
* CSS<br>
* Bootstrap

Installation
------------
Clone Github Repository:
```
$ git clone https://github.com/stephr3/wikipedia-clone.git
```
Open index.html file in your browser of choice.
